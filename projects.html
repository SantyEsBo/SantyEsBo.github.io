Course Registration Application

In this project, me and my team, composed of three other students, built an app based off SIS, the course site at UVA used to enroll for classes.

We added a few features and did not make an exact replica, we simply used the concept as a base for this small project.

The app was built in Java, using JavaFX, JSON, and SQL. It contained a database, HTML, and CSS implementations.

The apps features included user records (usernames and passwords), reviews of courses, review history, and add/delete courses with authorization.
